Privacy Policy

Last updated: 1 January 2026

This Privacy Policy describes how RapidReviewer (“we”, “us”, or “our”) collects and uses information when you use the RapidReviewer Chrome extension and the website at rapidreviewer.io (together, the “Service”).

Information We Collect via the Chrome Extension

The RapidReviewer Chrome extension is designed to help you log your own GitHub pull request activity for performance review and personal tracking. When you use the extension, it may collect:

  • A user‑provided API key, which you generate in your RapidReviewer account and paste into the extension options. This key is stored locally in your browser using Chrome’s chrome.storage API.
  • Basic pull request details that are already visible on GitHub pull request pages, such as:
    • Repository owner and name
    • Pull request number and title
    • Pull request URL
  • A short summary that you manually type when logging a pull request event (for example, describing what you did in that PR or review).

The extension does not collect:

  • Your general browsing history
  • GitHub passwords, access tokens, or cookies
  • Private messages or comments outside of the current PR page
  • Data from websites other than GitHub pull request pages

How We Use This Information

When you log a pull request event, the extension sends the information listed above to our backend API over HTTPS using your API key. We use this data to:

  • Create and maintain a private log of your work activity
  • Generate summaries that you can use for performance reviews or self‑reflection

We do not sell your data, and we do not share it with third parties for advertising or marketing purposes.

Storage & Security

  • Your API key is stored locally in your browser using Chrome’s storage mechanisms. You can remove or change it at any time from the extension options.
  • Data sent to our backend is transmitted over encrypted HTTPS connections.
  • We take reasonable steps to protect stored data, but no method of transmission or storage is 100% secure.

Remote Code

The RapidReviewer Chrome extension does not execute any remotely hosted code. All executable code runs from the packaged extension. Network requests made by the extension are limited to JSON API calls to the RapidReviewer backend.

Your Choices

  • You can uninstall the Chrome extension at any time from your browser’s extension settings.
  • You can remove or change your API key from the extension options at any time.
  • If you would like your account and associated data removed from our servers, you can contact us using the details below.

Changes to This Policy

We may update this Privacy Policy from time to time. When we do, we will update the “Last updated” date at the top of this page. Your continued use of the Service after any changes means you accept the updated policy.
